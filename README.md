# DevJobs - React

Plataforma de búsqueda de empleos construida con React y Vite.

## 🚀 Tecnologías utilizadas

- React 18
- Vite
- CSS Modules
- Fetch API + useEffect

## 🛠️ Funcionalidades

- Componentes reutilizables (JobCard, Header, Hero, Filters)
- Filtros combinados con estado en React
- Paginación con slice()
- Fetch de datos desde JSON
- Contador de resultados
- Diseño responsive
